Calculator Application - CodSoft Internship Task 2

Description

This project is a simple command-line Calculator developed using Python. It allows users to perform various arithmetic operations on two numbers through an interactive menu-driven interface.

Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Power
- Floor Division
- Continuous operation until the user chooses to exit
- Invalid input handling
- Division by zero protection

Technologies Used

- Python 3

How to Run

1. Install Python 3 on your system.
2. Save the program as "calculator.py".
3. Open a terminal or command prompt.
4. Navigate to the project directory.
5. Run the following command:

python calculator.py

Usage

- Select the desired operation from the menu.
- Enter two numbers when prompted.
- View the result.
- Continue performing calculations or choose the Exit option to close the program.

- Error handling for invalid inputs
